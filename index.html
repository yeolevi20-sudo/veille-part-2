Introduction

Aujourd’hui, le web et les applications mobiles font partie intégrante de notre quotidien. Créer un site ou une application demande d’utiliser des technologies web et des langages de programmation qui permettent de donner des instructions précises à l’ordinateur.

Ces technologies nous permettent de consulter des informations, communiquer, faire des achats, jouer, travailler… presque tout ce que nous faisons sur nos écrans !

I. La technologie web

1. Définition et rôle

La technologie web regroupe tous les outils et techniques permettant de créer et faire fonctionner un site ou une application sur Internet. Elle rend le contenu accessible sur ordinateur et téléphone, et permet de concevoir des sites interactifs et attractifs.

2. Les composants principaux

  • HTML (1991 – Tim Berners-Lee) : le squelette → structure le contenu (titres, paragraphes, images, liens…)
  • CSS (1996 – Håkon Wium Lie) : l’apparence → couleurs, polices, mise en page, responsive design
  • JavaScript (1995 – Brendan Eich) : l’interactivité → boutons, animations, formulaires intelligents, réactions aux clics

Ces trois langages travaillent ensemble : structure + design + comportement.

3. Importance

La technologie web est partout :

  • Consulter des informations
  • Communiquer (réseaux sociaux, messagerie)
  • Faire des achats en ligne
  • Utiliser des applications sur téléphone

II. Les langages de programmation web et mobile

1. Les langages web

  • HTML + CSS + JavaScript → base de tous les sites modernes

2. Les langages mobiles

  • Android : Java, Kotlin
  • iOS : Swift
  • Hybrides / multiplateformes : React Native, Flutter, Ionic...

3. Différence web ↔ mobile

  • Web : fonctionne dans le navigateur, facile à mettre à jour
  • Mobile natif : application installée, plus rapide, accès complet aux capteurs (caméra, GPS…)

API REST et GraphQL

Une API permet à deux applications de discuter entre elles.

  • REST : utilise des URLs et des méthodes (GET, POST, PUT, DELETE)
  • GraphQL : on demande exactement les données dont on a besoin (plus flexible)

Image simple : c’est comme commander au restaurant → tu demandes, le serveur te donne.

Développement mobile multiplateforme (Flutter, React Native)

Objectif : écrire un seul code qui fonctionne sur Android ET iOS.

  • Flutter → langage Dart (Google)
  • React Native → utilise JavaScript/React (Meta)

Avantage : gain de temps et d’argent.

Progressive Web Apps (PWA)

Un site web qui se comporte (presque) comme une vraie application mobile :

  • Peut s’installer sur l’écran d’accueil
  • Fonctionne hors connexion (service workers)
  • Notifications push
  • Chargement rapide

Conclusion

Les technologies web et les langages de programmation sont au cœur du monde numérique actuel. Ils évoluent très vite (IA, PWA, multiplateforme, sécurité renforcée…).
